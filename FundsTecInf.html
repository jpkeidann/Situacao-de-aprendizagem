Fundamentos De Tecnologia Da Informação:

S.A

A matéria de Fundamentos de TI apresenta os principais componentes físicos e lógicos de um computador, como hardware, software, sistemas operacionais, memória, sistema de arquivos e programas de escritório. É o estudo inicial para entender como computadores funcionam e como são utilizados no dia a dia de trabalho e tecnologia.



Hardware

Hardware se refere a todas as partes físicas de um computador — o que você pode tocar — e os componentes centrais dele incluem a placa-mãe, o processador (CPU) e a memória RAM. A placa-mãe é a base do sistema: nela todos os componentes internos se conectam para trocar dados e energia. O processador funciona como o “cérebro” — ele lê e executa instruções, realiza cálculos e coordena tudo o que o computador faz. A memória RAM é uma memória temporária onde o sistema armazena dados e programas em uso para acesso rápido — ela permite que o computador rode várias tarefas ao mesmo tempo com fluidez; quando desligamos o PC, os dados da RAM são apagados.

Sistema Operacional

O sistema operacional (SO) é o software essencial que faz a ponte entre o hardware e os programas: ele gerencia recursos como CPU, memória, dispositivos de entrada/saída e armazenamento, e provê interface para o usuário interagir com o computador. Exemplos comuns de sistemas operacionais são Windows, Linux (diversas distribuições), macOS e Android — cada um com características diferentes, mas todos cumprem a função de permitir o uso do hardware e rodar programas.

Sistema de Arquivos

O sistema de arquivos é a estrutura usada pelo sistema operacional para organizar e armazenar dados em disco (HD, SSD, pendrives etc.). Ele cuida de nomear arquivos e pastas, decidir onde os dados ficam armazenados, e permitir leitura, gravação, exclusão e busca de dados de forma organizada. Quando usamos o computador no modo texto (em um terminal ou “cmd”), interagimos com o sistema de arquivos via comandos: navegamos por pastas, listamos arquivos, movemos ou apagamos itens digitando instruções — o que dá mais controle, especialmente para quem sabe os comandos. Já em modo gráfico (como num explorador de arquivos), vemos pastas e arquivos como ícones e janelas, movemos itens com o mouse, clicamos para abrir, etc — interface mais intuitiva e visual. Essa diferença é apenas na forma de interação; por baixo dos panos, trata-se do mesmo sistema de arquivos.

Softwares de Escritório

Softwares de escritório são programas usados para escrita de textos, criação de planilhas, edição de tabelas, preparação de apresentações e documentos em geral — ferramentas comuns no dia a dia de estudos, trabalho e escritório. Por exemplo, Microsoft Word é usado para editar textos e relatórios; Microsoft Excel permite trabalhar com planilhas, cálculos automáticos e organização de dados; e Microsoft PowerPoint é usado para apresentações com slides. Esses programas ajudam a organizar informações, automatizar cálculos, formatar documentos e facilitar comunicação e documentação de ideias e resultados.
